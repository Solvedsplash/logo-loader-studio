# Logo Loader Studio

Create logo-based loading animations for mobile websites and web apps.

## Features

- Import any logo image and apply animation directly to the logo.
- Choose from `10,000` generated animation variants.
- Includes a curated default animation (`#7321`) tuned for logo loaders.
- One-click GIF export of the animated preview.

## Tech Stack

- Next.js (App Router)
- React
- `html2canvas`
- `gif.js`

## Run (Development)

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production

```bash
npm run build
npm run start
```
